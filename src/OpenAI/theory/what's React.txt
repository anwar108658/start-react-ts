1- what is React ?

React is a JavaScript library for building UI. Created by Meta. Instead of manually changing HTML using JavaScript - React lets you describe UI as components:


2. What is SPA?

SPA = Single Page Application.
Instead of reloading the entire page, only necessary UI updates.
Examples: 
    Facebook,
    Gmail,
    Netflix

3. What is JSX?

JSX is a syntax extension for JavaScript that allows developers to write UI code that looks similar to HTML. React converts JSX into JavaScript function calls.

4. Is JSX HTML?

No JSX looks like HTML but is converted into JavaScript.
